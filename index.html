New Business

Core Features Brainstorm

The team discussed and agreed on the following must-have features for the Pomodoro Timer:

  • Customizable work and break intervals
  • Audio notification when a session ends
  • Visual progress indicator
  • Task list integration so users can track what they worked on
  • Session history and basic analytics

Tech Stack Discussion

After some debate, the team settled on the following stack:

  • Frontend: HTML, CSS, and vanilla JavaScript
  • Testing: Jest for unit tests, Puppeteer for E2E
  • CI/CD: GitHub Actions

Bob suggested using React, but the team decided to stick with vanilla JS to keep things simple.
Carol volunteered to create initial wireframes by Friday, April 10th.

Sprint 1 Deadlines

The team agreed on the following deadlines for Sprint 1:

  • Wireframes complete — April 10
  • Basic timer functionality — April 14
  • Code review round 1 — April 16
